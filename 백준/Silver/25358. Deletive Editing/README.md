### 문제 설명

<p>Daisy loves playing games with words. Recently, she has been playing the following Deletive Editing word game with Daniel.</p>

<p>Daisy picks a word, for example, "<code>DETERMINED</code>". On each game turn, Daniel calls out a letter, for example, '<code>E</code>', and Daisy removes <strong>the first occurrence</strong> of this letter from the word, getting "<code>DTERMINED</code>". On the next turn, Daniel calls out a letter again, for example, '<code>D</code>', and Daisy removes its first occurrence, getting "<code>TERMINED</code>". They continue with '<code>I</code>', getting "<code>TERMNED</code>", with '<code>N</code>', getting "<code>TERMED</code>", and with '<code>D</code>', getting "<code>TERME</code>". Now, if Daniel calls out the letter '<code>E</code>', Daisy gets "<code>TRME</code>", but there is no way she can get the word "<code>TERM</code>" if they start playing with the word "<code>DETERMINED</code>".</p>

<p>Daisy is curious if she can get the final word of her choice, starting from the given initial word, by playing this game for zero or more turns. Your task it help her to figure this out.</p>
